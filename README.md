# The Fudge Kettle - Modern Website Redesign

A modern, high-conversion website redesign for The Fudge Kettle candy shop in Avon, Indiana.

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React Icons

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Features

- Modern, mobile-first responsive design
- Smooth page-load animations
- Scroll-based reveals
- Button hover micro-interactions
- Warm color palette (teal & chocolate)
- Premium typography (Playfair Display + Inter)
- Optimized for conversion

## Sections

1. **Hero** - Emotional headline with clear CTAs
2. **Product Highlights** - Image-forward product cards
3. **Why People Love Us** - Trust-building story section
4. **Custom Orders** - Event and gift emphasis
5. **Visit Us** - Location, hours, contact info
6. **Social Proof** - Press mentions and Instagram
7. **Footer** - Clean contact and navigation
